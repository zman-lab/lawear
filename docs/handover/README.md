# 법무사 학습매니저 — 프로젝트 개요

## 목적

법무사 2차 시험 PDF 자료를 TTS(Text-to-Speech) 텍스트로 자동 변환하는 파이프라인 구축.

## 대상 사용자

- 직장인 수험생 (출퇴근/운동 중 이어폰으로 TTS 학습)
- 눈으로 볼 시간이 부족 → 귀로 듣는 학습이 핵심
- 문제 구조 파악 + 답안 키워드 암기가 목표

## 시험 과목 (7과목)

### 논술 5과목
1. 민사소송법 (예비순환 미케)
2. 민법 (예비순환 미케)
3. 형법 (예비순환)
4. 형사소송법 (예비순환 모의고사)
5. 부동산등기법 (2순환 모의고사)

### 서류 2과목
6. 민사서류 (김지안 — 소장 작성)
7. 부동산등기서류 (기찬 — 등기신청서 작성)

## 핵심 발견 (분석 완료)

- 모든 PDF가 텍스트 임베딩 → OCR 불필요, AI가 바로 읽을 수 있음
- 서술형 자료(논술, 소장, 해설)는 TTS 매우 적합 → 전처리만 거치면 사용 가능
- 부등서류(등기신청서 양식)가 유일한 난관 → 표/2단배치/빈칸, 현재 보류 상태
- 밑줄/색상 강조는 전 과목 소실 → PyMuPDF 어노테이션 레이어 별도 추출 필요
- 직접 타이핑 대비 90%+ 자동화 가능 (부등서류 양식 부분만 수동 검수 필요)

## 검증 경위

- dev-airlock 프로젝트에서 PDF 분석 + TTS 변환 가능성을 사전 검증
- 7과목 PDF 파싱 품질 확인, 과목별 TTS 전처리 규칙 도출
- 논술 5과목 + 서류 2과목 샘플 추출 완료
- 사용자(수험생) 피드백 2차례 수렴 → 규칙 확정
- 이중 검증 팀 구조 설계 완료
